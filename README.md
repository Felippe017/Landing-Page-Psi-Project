## 🚀 Getting Started

### 📦 Install

```bash
# Clone the repository
git clone https://github.com/walber-vaz/template-react-vite-tailwind-ts

# Enter the project directory
cd template-react-vite-tailwind-ts

# Install dependencies
yarn install

# Start the project
yarn dev
```
